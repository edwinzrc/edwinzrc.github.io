fastMRI数据集的读取

PyTorch自定义数据集的读取

torch.utils.data.Dataset是一个抽象类，用户想要加载自定义的数据只需要继承这个类，并且覆写其中的两个方法即可：

  • ·__len __方法· :实现len(dataset)返回整个数据集的大小。
  • ·__getitem__方法· ：用来获取一些索引的数据，使dataset[i]返回数据集中第i个样本。

继承完这个类后，一般需要进行数据增强等操作，像kspace数据需要进行中心化等操作。最后使用第三方的库torch.utils.data.Dataloader。

PyQt5 DPI自适应

Posted on 2023-09-05 Edited on 2023-09-10 In UI

因电脑DPI的问题导致的问题

Bug

近期在尝试着用PyQt5+QtDesigner写一个医疗系统登录界面的UI，结果遇到这样的这样一个bug。直接进入正题，在QtDesigner中设计的如下图所示：

而运行出现出来的界面整体缩小了很多？？？？？

+

Debug

原因

网上找了很多都没能解决这个问题，后来经过一位大神帮助才解决了这个问题。因为本地电脑分辨率的问题，导致UI界面缩放以及字体的缺失。

解决

只要在main函数构造窗体对象前加入DPI自适应的代码就可以完美解决问题了。废话不多说，直接上代码。
QGuiApplication.setAttribute(Qt.HighDpiScaleFactorRoundingPolicy.PassThrough)
看一下解决后的UI界面。
